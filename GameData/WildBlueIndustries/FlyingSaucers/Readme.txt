Kerbal Flying Saucers: Build Flying Saucers in KSP!

Have you ever wanted to pretend that your kerbals found a crashed flying saucer and then reverse-engineered it? With Kerbal Flying Saucers (KFS), now you can! KFS is a mod that lets you research and build your own custom flying saucers that work within established gameplay mechanics while providing an exotic twist. ItÆs as easy as mixing and matching components to create a vehicle suited to your needs. And with its custom tech tree branch you can simulate your reverse-engineering efforts via the stock science system.
Kerbal Flying Saucers supports Pathfinder, MOLE, DeepFreeze, OSE Workshop, KIS/KAS, CTT, Snacks, TAC-LS, and more.

--RELEASE NOTES---

Changes

- Deprecated the Excalibur part set. This set is unfinished and won't be completed. The parts will be removed entirely at some point, so please plan accordingly.
NOTE: Existing craft will be unaffected, but you will no longer to be able to build the Excalibur in the editor.

- Added localization support to WBIGraviticEngine, WBIModuleGraviticRCS, and WBIPlumeController.

- Added localization support to all parts.

- Renamed Crazy Mode to Whiplash Mode.

- Removed Settings button from the hover control menu; functionality was deprecated some time ago...
NOTE: The hover control GUI is still able to be used to maneuver the craft.
NOTE: You can still use the RCS translation keys and the Brakes key to maneuver the craft when Whiplash Mode is active.

- You can now use the RCS up/down translation keys and the Brakes key to increase/decrease Hover Mode vertical acceleration and halt acceleration.
NOTE: Hover Mode vertical acceleration response will be disabled when Whiplash Mode is active.

- Added new Toggle Whiplash Mode action group action to the gravitic engine.

- Removed unused WBIGraviticLift part module.

- Consolidated functionality in WBIGraviticGenerator into WBIModuleGenerator, and removed WBIGraviticGenerator.

- Fixed issue where Toggle Hover Mode action wasn't appearing in the action groups list.
